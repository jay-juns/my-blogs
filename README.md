## 작업환경
1. 프론트 - react, hooks, redux, redux saga 
2. 서버 - Firebase firestore
<br />
<br />

## 앱의 용도/목적
- 포트폴리오 작업을 위해 글로벌 기업 앱(트위치, 유투브, 인스타그램, 넷플릭스 등등)의 일부 기능들을 스스로 직접 구현하기 위함입니다.

<br />
